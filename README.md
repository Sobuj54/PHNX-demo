# PHNX REPAIR | Frontier Mobile Engineering

PHNX REPAIR is a premium web platform for a high-precision mobile restoration ecosystem. The site showcases advanced repair capabilities, ranging from screen replacements to complex logic board micro-soldering, utilizing a modern design language and smooth interactive elements.

Live Demo: https://phnx-r.netlify.app/

## Features

- Immersive User Experience: Uses high-end animation libraries for fluid transitions and smooth scrolling.
- Responsive Design: Fully optimized for desktops, tablets, and mobile devices.
- Interactive Components: Includes an animated brand marquee, FAQ accordion, and step-by-step process visualization.
- Service Catalog: Detailed sections for various repair types, including smartphones, tablets, and gaming consoles.
- Reservation System: A dedicated interface for users to book diagnostic appointments.

## Tech Stack

- HTML5: Semantic structure.
- CSS3: Custom design system with glassmorphism and CSS variables.
- GSAP (GreenSock): Core animation engine for scroll-triggered events and UI transitions.
- Lenis: Smooth scrolling implementation.
- Google Fonts: Plus Jakarta Sans for typography.

## Getting Started

### Prerequisites

A modern web browser (e.g., Chrome, Firefox, Safari, or Edge).

### Installation

1. Clone the repository:
   git clone https://github.com/Sobuj54/PHNX-demo.git

2. Navigate to the project directory:
   cd phnx-repair

3. Open index.html in your browser or run it through a local development server.

## Usage

- Navigation: The fixed navbar provides quick access to the Services, Process, and FAQ sections.
- Interaction: Scroll through the "The Workflow" section to see the animated stacking card effect.
- Booking: Fill out the form in the "Reserve Your Slot" section to submit a repair request.
